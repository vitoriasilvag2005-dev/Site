O que é Docker?

Docker é uma plataforma que permite criar, distribuir e executar aplicações dentro de containers. Um container é como uma "caixinha" isolada que contém tudo o que a aplicação precisa para rodar, incluindo código, bibliotecas e dependências.

Isso garante que a aplicação funcione da mesma forma em qualquer lugar: no seu computador, no servidor da empresa ou na nuvem.

Exemplo rápido:

Imagine que você tem um site que precisa de uma versão específica do Python. Com Docker, você cria um container que já vem com essa versão do Python instalada, evitando conflitos com outras aplicações.
