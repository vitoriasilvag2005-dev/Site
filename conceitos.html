O que é Docker?

Docker é uma plataforma que permite criar, gerenciar e executar aplicações em containers isolados. Um container é uma unidade leve, portátil e consistente que empacota código, runtime, bibliotecas e dependências.

Diferença entre container e máquina virtual:

  • Containers compartilham o mesmo kernel do sistema operacional, tornando-os mais leves.
  • Máquinas virtuais incluem sistema operacional completo, consumindo mais recursos.

Componentes principais do Docker:

  • Docker Engine: Motor que cria e executa containers.
  • Docker Hub: Repositório de imagens públicas e privadas.
  • Dockerfile: Arquivo de instruções para criar imagens personalizadas.

Exemplo prático:

# Criando uma imagem personalizada a partir de um Dockerfile
docker build -t minha-imagem .
