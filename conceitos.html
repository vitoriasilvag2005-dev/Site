Conceitos de Docker

Início Conceitos Vantagens

Containers

Um container é como uma pequena caixa isolada onde sua aplicação roda com tudo que precisa (bibliotecas, dependências), sem interferir no restante do sistema.

Imagens

Imagens Docker são como “modelos” de containers. Você cria o container a partir de uma imagem, garantindo consistência.

Dockerfile

O Dockerfile é um arquivo de instruções que define como a imagem deve ser construída. Exemplo:

FROM nginx:alpine
COPY *.html /usr/share/nginx/html/
COPY style.css /usr/share/nginx/html/
EXPOSE 80
